# Инструкция по рабшоте с разметкой Markdown

## Выделение текста 

Чтобы выделить текст курсивом необходимо выделить его  (*). Например *вот так*.

Чтобы выделить текст полужирным необходимо выделить его (**). Например **вот так**.
